## Optional Surface Projection Smoothing - 2026-05-20

Implementation:

```text
Added optional export-time surface smoothing as an explicit command/setting, default off.

Behavior:
- Does not run automatically.
- Projects existing mesh vertices back to the SDF zero surface using finite-difference SDF normals.
- Clamps each projection step to a fraction of the export cell size.
- Only affects vertex positions; it does not alter triangle connectivity.
- Runs topology cleanup again after projection.
```

## Surface Smoothing Non-Regression Guard - 2026-05-20

Implementation:

```text
Added smooth_export_mesh_with_topology_guard(...).

The smoothing path now:
1. Analyzes the already-clean baseline mesh.
2. Projects vertices to the SDF zero surface.
3. Re-runs export topology cleanup.
4. Analyzes the cleaned smoothed mesh.
5. Keeps the smoothed mesh only if its bad-edge score is not worse than baseline.
```
